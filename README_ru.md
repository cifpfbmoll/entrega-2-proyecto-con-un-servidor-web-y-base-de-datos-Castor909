# Проект с веб‑сервером и сервером базы данных

## Упражнения для повторения

1. Скачать следующие образы: `ubuntu:24.04`, `httpd`, `tomcat:9.0.39-jdk11`, `jenkins/jenkins:lts`, `php:7.4-apache`.
2. Показать образы, которые вы загрузили.
3. Создать демон‑контейнер (контейнер в фоновом режиме) с образом `php:7.4-apache`.
4. Проверить размер контейнера на диске.
5. С помощью инструкции `docker cp` можно копировать файлы в контейнер или из контейнера. Можно найти информацию на этой странице: https://docs.docker.com/engine/reference/commandline/cp/.
   Создайте файл на вашем компьютере со следующим содержимым:

```php
<?php
echo phpinfo();
?>

```

   Скопируйте файл `info.php` в каталог `/var/www/html` контейнера с помощью `docker cp`.
6. Снова проверьте пространство, занимаемое контейнером.
7. Откройте файл `info.php` в веб‑браузере.

## Описание задания

### Веб‑сервер

* Запустить контейнер с образом `php:7.4-apache`, назвать его `web` и сделать доступным с вашей машины на порту 8000.
* Поместить в корневой каталог веб‑сервиса (`/var/www/html`) этого контейнера файл `index.html` со следующим содержимым:

```html
<h1>Привет, я XXXXXXXXXXXXXXX из IFC33X</h1>
```

  Вы должны заменить `XXXXXXXXXXX` на ваше имя и фамилии.

* Поместить в тот же корневой каталог файл `index.php` со следующим содержимым:

```php
<?php echo phpinfo(); ?>
```

* Для создания файлов у вас есть три альтернативы:
  * Выполнить bash в контейнере интерактивно и создать файлы там.
  * Выполнить команду `echo` в контейнере с помощью `docker exec`.
  * Использовать `docker cp`, как показано в упражнении 5.

### Сервер базы данных

* Запустить контейнер с именем `bbdd`, который будет работать на образе `mariadb`, и сделать его доступным на порту 3336.
* Перед запуском посетите страницу образа на Docker Hub: https://hub.docker.com/_/mariadb и установите необходимые переменные окружения, чтобы:

  * Пароль root был `root`.
  * При запуске автоматически создавалась база данных с именем `prueba`.
  * Создавался пользователь `invitado` с паролем `invitado`.
